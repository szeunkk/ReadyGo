핵심요구사항) 다음의 조건에 맞게 파티 필터 기능을 step-by-step 으로 구현하고, 적용 결과를 체크리스트로 반환할 것.
            1) 유저시나리오(장르 필터하기)
                1. 필터선택박스 클릭
                2. 선택 가능 메뉴가 party.tsx의 genreItems와 일치하는지 확인
                    - 전체
                    - 액션
                    - 모험
                    - 캐주얼
                    - 얼리 액세스
                    - 무료 플레이
                    - 인디
                    - MMO
                    - 성인 요소
                    - 레이싱
                    - RPG
                    - 시뮬레이션
                    - 스포츠
                    - 전략
                    - 폭력성
                3. 메뉴 선택시, 선택한 장르와 일치하는 파티 카드만 노출되는지 확인
                    - 필터링 로직: 
                        1) party_posts.game_title과 steam_game_info.name을 부분일치로 매칭하여 해당 게임 정보 조회
                           - 부분일치: game_title이 name에 포함되거나, name이 game_title에 포함되는 경우
                           - 대소문자 구분 없이 매칭
                        2) 선택한 장르의 id 값(영어, 예: 'Action', 'Violent')이 steam_game_info.genres 배열에 포함되어 있는지 확인
                        3) 포함되어 있으면 해당 게임의 파티만 표시
                    - "전체" 선택시(id가 'all'): 모든 파티 표시
                    - NULL 처리:
                       - steam_game_info에 해당 게임이 없는 경우: 장르 필터에서 제외 (해당 장르로 필터링 시 표시되지 않음)
                       - steam_game_info.genres가 NULL인 경우: 장르 필터에서 제외 (해당 장르로 필터링 시 표시되지 않음)
                    - 예시: 'Violent' 선택시, steam_game_info.genres에 "Violent"가 포함된 게임의 파티만 표시

            2) 유저시나리오(게임 이름 검색하기)
                1. 검색창에 게임 이름을 입력하여 검색
                2. 검색된 결과 확인
                    - 검색 로직: 
                        1) party_posts.game_title에 검색어가 포함되어 있는지 확인
                        2) party_posts.game_title과 steam_game_info.name을 부분일치로 매칭하여, steam_game_info.name에 검색어가 포함되어 있는지 확인
                        3) 위 두 조건 중 하나라도 만족하면 해당 파티 표시
                    - 대소문자 구분 없이 부분 일치 검색
                    - NULL 처리: steam_game_info에 해당 게임이 없어도 party_posts.game_title에서 검색되면 표시됨

            3) 유저시나리오(검색결과 필터하기)
                1. 검색창에 게임 이름을 입력하여 검색
                2. 검색된 결과를 바탕으로 필터선택박스를 변경
                    - 선택한 장르와 일치하는 파티 카드만 노출되는지 확인
                    - 필터링 로직: 
                        1) 검색 결과 중에서 party_posts.game_title과 steam_game_info.name을 부분일치로 매칭
                        2) 선택한 장르의 id 값(영어)이 steam_game_info.genres 배열에 포함되어 있는지 확인
                        3) 검색 조건과 필터 조건을 모두 만족하는 파티만 표시
                    - 검색과 필터가 동시에 적용되어야 함 (AND 조건)
                    - NULL 처리:
                       - steam_game_info에 해당 게임이 없거나 genres가 NULL인 경우: 장르 필터에서 제외
                       - 검색은 party_posts.game_title에서도 수행되므로, 게임 정보가 없어도 검색 결과에는 포함될 수 있음
                       - 하지만 장르 필터가 적용되면, steam_game_info.genres에 해당 장르가 포함된 파티만 표시됨
